# Estruturas Condicionais - Python 🐍

## Operadores de comparações:


| Operadores | Significado |
|----------|:---------|
| ==  | Igual a  |
| !=  | Diferente de  |
| > | Maior que|
| < | Menor que|
| >= | Maior ou igual que |
| <= | Menor ou igual que |

# Exemplo dos comparadores:

```
a = 10
b = 20
print(a == b)  # Falso
print(a != b)  # Verdadeiro
print(a < b)   # Verdadeiro
print(a >= 10) # Verdadeiro


```
