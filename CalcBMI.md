# Atividade 04 - Cálculo de índice de massa corporal (IMC) com Python 🐍
```
def Main():
  height = float(input("Digite sua altura em centimetros: "))
  weight = float(input("Digite seu peso em quilos: "))
  imc = weight / (height * height)
  print(f"Seu IMC é: {imc}")
if __name__ == "__main__":
  Main()
```
