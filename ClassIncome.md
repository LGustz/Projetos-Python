# Atividade 02 - Classificação de renda com Python 🐍
```
salario = float(input("Digite o seu salário: "))

if salario < 1400:
  print(f"Situação de risco!\n Salário: {salario}")
elif salario < 2799:
  print(f"Renda baixa!\n Salário: {salario}")
elif salario < 6999:
  print(f"Renda média!\n Salário: {salario}")
else:
  print(f"Renda alta!\n Salário: {salario}")
```
