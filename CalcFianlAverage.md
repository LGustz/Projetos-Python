# Atividade 03 - Calcular a média final com Python 
```
def Main():
  note01 = float(input("Digite a nota do teste 01: "))
  note02 = float(input("Digite a nota do teste 02: "))
  note03 = float(input("Digite a nota do teste 03: "))
  note04 = float(input("Digite a nota do teste 04: "))
  average = (note01 + note02 + note03 + note04) / 4
  print(f"Média final: {average}")
if __name__ == "__main__":
  Main()
```
